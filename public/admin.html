Importar desde WordPress / LearnPress

Trae automáticamente los cursos de flordechanar.cl a esta plataforma.

1

Importar cursos (sin configuración)

Importa los 13 cursos conocidos con título, precio e imagen. No necesitas hacer nada previo — solo haz clic.

2

Importar lecciones y contenido real

Para importar el contenido interno de cada curso necesitas una Contraseña de Aplicación de WordPress.

Cómo obtenerla (2 minutos):
1. Ve a flordechanar.cl/wp-admin → Tu Perfil
2. Baja hasta "Contraseñas de aplicación"
3. Escribe el nombre Migracion → clic en Añadir
4. Copia la contraseña que aparece (solo se muestra una vez)
5. En EasyPanel → tu servicio → Variables de entorno → agrega:
   WP_USER = tu_usuario_wordpress
   WP_APP_PASS = la contraseña copiada
6. Guarda y haz Deploy → luego vuelve aquí y haz clic en el botón de abajo
3

Migrar estudiantes y matrículas

Los usuarios de WordPress usan un sistema de contraseñas distinto. El proceso es:

Pasos en phpMyAdmin de WordPress:
1. Ve al panel de hosting → phpMyAdmin → selecciona la BD de WordPress
2. Clic en SQL (pestaña de arriba)
3. Pega y ejecuta esta consulta para ver todos los estudiantes: 
SELECT u.ID, u.display_name AS nombre, u.user_email AS email,
       u.user_registered AS registrado
FROM wp_users u
JOIN wp_usermeta um ON u.ID = um.user_id
WHERE um.meta_key = 'wp_capabilities'
  AND um.meta_value LIKE '%subscriber%'
ORDER BY u.user_registered;
4. Exporta el resultado como CSV
5. En esta plataforma ve a Usuarios → Crear usuario por cada estudiante
   (La contraseña temporal recomendada: FlordeChanar2025!)
6. Notifica a cada estudiante para que cambie su contraseña
Tip: Si tienes muchos estudiantes, comparte este link con ellos para que se registren solos: https://tu-dominio.com/login.html Luego ve a Inscripciones → inscribe manualmente cada estudiante en su curso correspondiente.
